# jeo-party

## Description
Play jeopardy with friends!

## Installation

1. Open VS Code Terminal
2. Enter `npm i` into the command line
3. Run `mysql -u root -p`
4. Run `SOURCE schema.sql`
5. exit mysql
7. run `node index.js`
